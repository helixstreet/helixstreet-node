# helixstreet.io - the life sciences blockchain

helixstreet.io is a blockchain focused on life sciences applications including genomics and proteonomics. the chain allows the complete proof of ownership of genomic data and the relationship between genomic data. Helixsstreet provides a  single, immutable record where people can constantly contribute and draw connections between sets of data. Every suggested connection (or set of connections if enough inferences can be drawn by the two contributors) would have to be validated by both providers of the ancestry information, then further approved through on chain governance.
furthermore the helixstreet.io blockchain supports helixgroups to make drug development and rating more objective.
